## 博彩商城项目

### 技术栈

1.后端

- 核心框架:  Spring Boot
- 视图框架:  Spring MVC 
- 持久层框架：Spring Data JPA
- 模板引擎:  Thymeleaf
- 项目构建工具:  Gradle
- 工具类:   Apache Commons 、 fastjson
- 效率开发:  lombok、devtools
- 短信验证:  阿里大于

2.前端

- JS框架：jQuery 
- CSS框架：Twitter Bootstrap 

3.数据库

- Mysql 5.5

4.开发工具

- Intellij IDEA 2018.1.2
